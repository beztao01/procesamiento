Configuración Render


Para este proyecto vamos a usar un modelo 3d que se encuentra en la libreria de SketchFab, el susodicho es el Lamborghini Gallardo LP560.

Es un archivo que se encuentra comprimido, en dos carpetas una llamada source y otra texture, en el source se encuentra nuevamente comprimida otra carpeta llamada gallardo, hay que descomprimirla y usamos ahora el archivo gallardo.glb, que es el que tiene todas las texturas preparadas.

El objetivo de esta práctica es configurar el motor de render cycles, los primeros pasos son entrar en edit | preferences |system, y dependiendo de la tarjeta gráfica activarla, Nota: Siempre y cuando el equipo tenga una, si no se puede trabajar con los graficos integrados que tiene el procesador.
